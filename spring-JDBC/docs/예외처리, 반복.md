# 스프링과 문제 해결 - 예외처리, 반복

## 체크 예외와 인터페이스

- 서비스 계층은 순수하게 유지하는게 좋다. 예외에 대한 문제도 마찬가지다.
- `SQLException` 에 대한 의존을 제거하려면 어떻게?
- 서비스가 처리 불가능하므로 체크 → 언체크 변경해서 서비스로 던짐
- 서비스에서 해당 예외를 무시할 수 있고, 순수한 코드 유지 가능.
